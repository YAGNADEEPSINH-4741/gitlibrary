# GitBook - The Library of Code

A dynamic, interactive web application that acts as a living catalog for GitHub repositories.

## Features
- Search through GitHub repositories instantly.
- Beautiful, responsive UI that looks great on mobile and desktop.
- 3D library visualization.
- Detailed repository genome and statistics.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Technologies
- React
- Vite
- TypeScript
- CSS
- Framer Motion
